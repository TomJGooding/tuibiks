# tuibiks

Rubik's Cube in the terminal

![gif](https://raw.githubusercontent.com/TomJGooding/tuibiks/main/assets/tuibiks.gif)

> [!NOTE]
> The cube is rendered using Symbols for Legacy Computing (SLC).
> Some terminal emulators seem to render SLC better than others.
> The kitty terminal renders a perfect cube, as shown in the GIF above.
